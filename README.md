# Lodowisko

 **Krótko:** Platforma webowa do kompleksowej obsługi lodowiska miejskiego. System jest podzielony na dwie części: kliencką (rezerwacje, zakupy) oraz administracyjną (zarządzanie zasobami).

---
### Kluczowe funkcjonalności:
* System Rezerwacji: Wybór konkretnej daty i godziny.

* Wypożyczalnia Sprzętu: Możliwość zarezerwowania łyżew (według rozmiaru).

* Panel Administratora: Podgląd liczby osób na lodowisku w danym czasie (limity bezpieczeństwa), zarządzanie cennikiem i stanem magazynowym wypożyczalni.

* Harmonogram: Kalendarz z zaznaczonymi przerwami technicznymi (czyszczenie lodu) oraz rezerwacjami pod zajęcia nauki jazdy na łyżwach.
